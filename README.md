# SwingTrader 2026

A responsive web application for swing trading with 5 main navigation tabs.

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop
- **5 Navigation Tabs**: Dashboard, Portfolio, Analytics, Strategies, Settings
- **Mobile Menu**: Hamburger menu for mobile devices
- **GitHub Pages Ready**: Deploy directly to GitHub Pages

## Navigation

- **Dashboard**: Main trading dashboard
- **Portfolio**: View and manage trading portfolio
- **Analytics**: Trading analytics and performance metrics
- **Strategies**: Trading strategies management
- **Settings**: Application settings

## Getting Started

1. Enable GitHub Pages in repository settings
2. Point to the `main` branch
3. Access your app at: `https://natalivm.github.io/SwingTrader2026/`

## Technologies Used

- HTML5
- CSS3 with CSS Variables
- Vanilla JavaScript
- Responsive Design

## File Structure

- `index.html` - Main HTML file
- `styles.css` - Styling and responsive design
- `script.js` - Navigation and interactivity
- `.nojekyll` - GitHub Pages configuration
